Printers

        
Each Zebra printer is enrolled in the same Zebra Cloud account; add it here by name + serial. Workers pick which printer to print to on the Label Printer screen.

        

        
Add a printer

        

          

          

        

        

          
          
        

        

          How to add a new Zebra printer (step by step)
          
    
            
  1. Power on the new Zebra and connect it to the warehouse Wi-Fi.
    2. 
            
  2. Enroll it in Zebra Cloud (one time): on a PC, open Zebra Setup Utilities → select the printer → Open Communication With Printer, paste the line below, click Send, and let the printer reboot:
              
    {}{"weblink.ip.conn1.location":"https://savanna-device.zpc.zebra.com/weblink/connect?r=6cc4e1a77bafa54e37c597155d0eb2e1","device.reset":"1"}
    
            
    3. 
            
  3. Find the printer's serial number — on the label, or print a config label (Menu → Print → Settings).
    4. 
            
  4. Enter a name and that serial in the boxes above, then tap Add printer. It shows up in the printer toggle automatically.
    5. 
          

          
Every printer uses the same Zebra Cloud account, so you only enter a name + serial here — no API key per printer. Add as many as you like.

        

        


        
Each different part number sent to the printer = 1 cloud call (the quantity/copies are free). Zebra's free tier is 100 calls/day; beyond that they bill ~$0.01 each. Set a daily cap to stop printing once a number of calls is reached.
